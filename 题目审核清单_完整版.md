### bk_cast_1
- **题型**: code_fill
- **难度**: ☆☆☆☆☆
- **知识点**: 类型转换, int

**技术描述**:
将字符串 "99" 转为整数并加 1，打印结果。

**初始代码**:
```python
# 将 "99" 转为整数，加 1 后打印
print(int("") + 1)
```

**正确代码**:
```python
print(int("99") + 1)
```

**预期输出**: `100`

**解析**: int() 将字符串转为整数。字符串必须表示有效的整数。

**叙事解析**: ❌ 缺失
