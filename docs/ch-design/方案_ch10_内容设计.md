# ch10 内容设计：async/await、uv 与打包发布

---

## 一、为什么学这些

| 知识点 | 为什么必须学 | 不学的后果 |
|--------|------------|-----------|
| async/await | 现代 Python 的异步编程基础，Web/API/爬虫的核心 | 看不懂异步代码，无法处理 IO 密集型任务 |
| uv | 下一代 Python 包管理器（Rust 实现），替代 pip + venv | 不知道现代 Python 生态的进化方向 |
| 打包 | 写了一个好库怎么给别人用？setup.py/pyproject.toml | 写好代码只能自己用，无法发布 |

**注意**：uv 是 Rust 工具（Pyodide 不可运行），打包发布也是概念为主。
