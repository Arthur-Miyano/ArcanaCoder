## 二、精准重试机制

### 定义

| 概念 | 说明 |
|------|------|
| `retryIds` | string[]，待重试的错题 questionId 列表 |
| `isRetryMode` | boolean，为 true 时强制用户必须答完当前错题 |
| 单题重试 | 从结果总览点击某错题，进入那一道题的修正模式 |
| 批量重试 | 从结果总览点击"重试所有错题"，依次遍历 retryIds |

### 重试规则

```
答对 → 从 retryIds 移除 → 
  若 retryIds 为空 → 检查是否全对 → 全对则通关
  若 retryIds 仍有剩余 → 
    模式为"单题"→ 回到结果总览
    模式为"批量"→ 跳到下一道错题
答错 → 不跳题，留在当前题 →
  显示 FeedbackPanel（和正常答题一样显示错误原因）
  用户修改代码后可以再次提交
  禁用"下一题"按钮（不显示），只显示"提交答案"
  直到答对才能离开
```

### 退出保护

- 重试模式下，`emit('back')` 前检查 `retryIds.length`
- 若 `retryIds` 非空，弹出确认："还有 {n} 道错题未重试，确定退出吗？"
- 确认退出后，`retryIds` 和 `isRetryMode` 重置为初始状态

## 三、输出 diff 增强

### diff.ts 只做输出对比

变量名对比从 `diff.ts` 移除，改为 `codeValidator.ts`。

**codeValidator 不做完整语法检查**——Python 解释器已经通过 errorParser 翻译成中文显示了。codeValidator 只做最小粒度的关键变量名提示：

| 检查项 | 实现 | 失败反馈 |
|--------|------|---------|
| 是否包含期望的关键变量名（如 area） | 正则 `\barea\b` 排除字符串/注释 | "提示：建议使用 area 变量存储计算结果" |
| 是否包含 return（若要求函数） | 正则 `\breturn\b` | "提示：函数需要 return 返回值" |

### diff 分类修正

原方案：去掉标点数字后一致 → format。太粗糙。

修正：提取数字序列后转为数值比较，不是字符串比较。

```
1. 提取两个输出中的所有数字（含小数点），转为 float 逐位比较
   if (数值不完全一致) → logic（数字不同，逻辑有误）
2. 若数值完全一致 → 比较去除数字后的文本部分：
   if (文本一致) → format（纯格式/标点/空格差异）
   if (文本不一致) → logic（数值相同但内容不同）
```

**format 类型举例：**
- `78.5` vs `78.50` → 数值 78.5 == 78.5 → 去除数字后 `` vs `` 空串一致 → **format** ✅
- `面积：78.5` vs `面积: 78.5` → 数值一致，去除数字后 `面积：` vs `面积:` → **format**（标点差异）
- `面积78.5` vs `周长78.5` → 数值一致，去除数字后 `面积` vs `周长` → **logic**（内容不同）
- `78.50` vs `78.5` → 数值一致 → **format**（尾部零差异）

### FeedbackPanel 显示

左栏下方增加差异面板：

```
┌─ 差异分析 ─────────────────────────┐
│ 你的输出：面积是78.5                 │
│ 期望输出：面积是 78.5                │
│ 类型：格式差异                        │
│ 差异点：                             │
│   第 4 个字符 后缺少一个空格          │
└────────────────────────────────────────┘
```

若类型为 logic，不显示"逻辑正确"提示（因为逻辑确实有误）。

## 四、输出格式教育

在诺克斯分级提示中增加一档：

| 尝试次数 | 诺克斯提示 |
|----------|-----------|
| 首次 hint | 正常提示 |
| 答错 1 次（hintRoleplay） | 角色口吻提示 |
| 答错 2 次（hintDirect） | 直接告诉答案，末尾加一句话强调格式 |
| **答错时 diff 为 format 类型** | **额外提示："你的计算是对的，但输出格式与题目要求不完全一致。Python 对字符串的精确匹配包括标点符号和空格。"** |
